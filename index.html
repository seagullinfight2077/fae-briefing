1帳號登入失敗（帳號問題 - 1,120 件，佔比最高）
問題描述

站長回報玩家無法登入，或帳號顯示被鎖定、被禁用。LINE 站中帳號問題佔所有客訴 26%，為最頻繁問題類型。

回覆話術
```
請稍等，立即為您查詢中
為了快速定位問題，麻煩提供：
1. 受影響玩家帳號
2. 最後一次成功登入時間
3. 錯誤訊息截圖
收到後立即查詢，有結果第一時間回覆您
```
注意事項
- 若站長反映「很多玩家」登不進去，需判斷是否為集體事件，立即通知當班幹部 - 帳號被鎖定通常為登入失敗次數過多或風控觸發，需在後台確認原因後再回覆 - 不要告知鎖定機制的內部規則（例如「連錯 5 次」） ---
2優惠活動條款爭議（優惠活動爭議 - 843 件）
問題描述

站長質疑活動獎勵未發放、條款與理解不符，或要求修改活動規則。伍洲站(L09)此類問題最多（152 件）。

回覆話術
```
請稍等，立即為您查詢中
為了精準確認，麻煩提供：
1. 活動名稱
2. 受影響玩家帳號
3. 您方認為的差異點
我們將核對活動頁面條款，確認後同步給您
```
注意事項
- 以條款原文為準，只引用不解釋背後邏輯 - 站長若要求修改活動規則，回覆「這部分需要跟相關單位確認」，不可自行答應 - 禁止使用「其他站長都接受」等社會認同話術，容易引爆情緒 ---
3遊戲串接與上架問題（遊戲上架/串接 - 578 件）
問題描述

站長要求新增遊戲、詢問上架進度，或反映遊戲串接後無法正常運作。吉滿滿(L06)與伍洲(L09)為此類問題最多的站點。

回覆話術
```
請稍等，立即為您查詢中
關於遊戲上架進度：
1. 已確認該遊戲商是否已完成串接
2. 目前正在排查 [語系/ICON/賠率] 設定
3. 確認後立即回覆您最新狀態
如需確認更多細節，麻煩提供遊戲名稱及遊戲商名稱
```
注意事項
- 先確認該站是否已串接該遊戲商，避免不必要的期待 - ICON 素材先查 Google Drive，沒有再找遊戲商 - 上線前必須通過四項檢查：語系、賠率限紅、注單回傳、可下注 ---


! FAE 鐵律（必讀）

五大鐵律 — 凌駕所有技術判斷

  1. 先自查再問人 — 查本文件 → FAQ → 歷史對話 → 後台 UAT / Postman → Kibana → 再找工程
  2. 禁止時間承諾 — 不說「幾分鐘」「幾小時好」→ 用「有進度第一時間同步」
  3. 禁止絕對用語 — 不說「一定」「絕對」「保證」→ 用「依目前資訊來看…」
  4. 一次問完不擠牙膏 — 把需要的資料一次列齊，禁止連續追問
  5. 以條款為準，只引用不解釋 — 活動爭議引用原文，不自行詮釋

禁用話術

禁止說法原因
「一定能解決」「保證沒問題」無法保證結果
「預計 30 分鐘內完成」時間不可控
「這是工程的問題」禁止甩鍋
「之前也是這樣處理」每次需獨立確認

建議話術（進度承諾）

建議說法適用場景
「已收到，確認中，有進度第一時間同步」所有問題通用開場
「依目前資訊來看，可能是 XX 原因」需推測原因時
「已回報相關單位，目前等待回覆」需升級時
「麻煩提供 XX，我們加速處理」需補充資料時
